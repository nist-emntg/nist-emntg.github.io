What is this page about?

This page presents some research artifacts that were spawn from the Secure Smargrid project with the Emerging & Mobile Network Technologies Group (EMNTG) at NIST. We believe that sharing our code on GitHub we will make it more accessible and foster more contribution. Because the code derives from our research, it should be considered as experimental and is sometimes incomplete. Also, some contribution are meant to be used in conjunction.

Please note that this page is not the official page of the Emerging & Mobile Network Technologies group at NIST. It only presents a small subset of our group's contributions and efforts. People interested to learn more about our group can go on our official website.

Projects

IEEE 802.15.4 networks emulation

wiredto154

A IEEE 802.15.4 PHY layer emulator over Ethernet links

This is an emulator for IEEE 802.15.4 networks over Ethernet links. The emulator takes a node map as an input and computes for each packets it receives which nodes in the simulation should actually receive it. Frame are encapsulated within UDP messages. The emulator is optimized for speed and tries to provide real time emulation of IEEE 802.15.4 networks.

The emulator currently only works with modified Contiki "native" nodes. Connector for Linux nodes is possible but has not be written yet.

wiredto154-logger

Logging tool for wiredto154 simulations

Nodes participating in a wiredto154 can use the emulator's facility to report logging message. The logger will read them in real time. The logger also contains a real time viewer that indicates the status of the nodes.

Implementation of the RPL routing protocol (RFC 6550)

simpleRPL

A linux-based implementation of the Routing Protocol for Low-Power and Lossy Networks (RPL)

This is an implementation of the Routing Protocol for Low-Power and Lossy Networks. This routing protocol has been designed for low power nodes with low memory capacity.

This implementation can operate across multiple interfaces (where links can be of different type), which makes it useful for bridge-like devices.

Routing

a libnl-route binding for python

One of the support library of simpleRPL. It is a python wrapper for the libnl (a netlink library). It enables the python code to add and remove networks routes, addresses, etc.

RplIcmp

A python module for handling ICMP messaging for the RPL protocol

One of the support library for simpleRPL. It is a python wrapper for sending and receiving ICMPv6 messages.

Linux related code

virtual-ieee802154-serial

Virtual tunnel interface for the IEEE 802.15.4 serial protocol

Certain IEEE 802.15.4 devices, like the RedBee Econotag, present a serial-like interface for the Linux kernel to connect to. This software provides a virtual serial interface, so that the Linux kernel can setup a virtual IEEE 802.15.4 device on top of it. Once set up, all the traffic that go though this interface is encapsulated into UDP packets.

linux802154-regression-tests

Regression tests for the Linux IEEE 802.15.4 and 6LoWPAN stack

This basic regression tests suite ensures that new patches within the Linux kernel do not introduce regressions on basic features IEEE 802.15.4/6LoWPAN features.

Works related to the RedBee Econotag

For our testbed, we use the RedBee Econotag. We wrote some code in order to better manage the device.

serialreadts

Serial port reading with timestamp information (serialreadts)

When reading logs from the Econotag serial interface, it is sometimes useful to see exactly when a message occurred. This small tool append a timestamp at the beginning of each new lines thus making reading logs very easy.

linux802154-serialdev

Friendly fork of Mariano Alvira's linux serial driver firmware for the Redbee Econotag

A slightly modified version of the RedBee Econotag firmware that works along a modified IEEE 802.15.4 serial driver for Linux. It activates extra feature, such as the auto-acknowledgment.

Contiki related code

ecc-light-certificate

A simple ECC certificate library

A small certificate library for sensor networks. This library aims to minimize the RAM and ROM footprint as well as the energy consumption. It uses a non-standard but very compact and lightweight certificate format.
