What is this page about?

This page presents some research artifacts that were spawn from the Secure Smargrid project with the Emering & Mobile Network Techonologies group (EMNTG) at NIST. We believe that sharing our code on GitHub we will make it more accessible and foster more contribution. Because the code derives from our research, it should be considered as experimental. Also, some contribution are meant to be used together.

Please note that this page is not the official page of the Emerging & Mobile Network Technologies group at NIST. It only presents a small subset of our group's contributions. People interested to find more about our group can go on our official website.
